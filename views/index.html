Playdate SDK API Reference

D bindings to the Playdate SDK as an idiomatic D wrapper around the API.

Version Relation to Playdate SDK

Published versions of this library are guaranteed to align with the Playdate SDK's MAJOR and MINOR versions. For example, version 1.12.3 of the library aligns with 1.12.x versions of the Playdate SDK.
